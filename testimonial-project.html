Testimonial Grid

next project >

This project had a slight learning curve as this was my first introduction to Tailwind CSS. To get things started I used the CDN and read through the documentation and after a short while the way it worked made complete since.

From my time working on this project I understand the appeal of Tailwind CSS, it's quick easy and allows you to style whatever component all in the same html file. There are a ton of things I still have left to learn within Tailwind CSS but overall I am happy with the outcome and I plan on using Tailwind CSS for future projects to become more proficient.

The Goal

The goal of this project was to recreate a testimonial grid design showcasing various input from customers in different ways.

Built With
